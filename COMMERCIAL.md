# 商用利用窓口

reader-facing な本文・図版・付録など、`LICENSE-SCOPE.md` で `CC BY-NC-SA 4.0` と整理したコンテンツの商用利用は、別途契約の対象です。

## 対象

- 研修教材、社内教育コンテンツ、営業資料、商用出版物への転載
- 有償サービス・有償講座・企業向け配布物での利用
- `CC BY-NC-SA 4.0` の非営利条件を満たさない利用

## 対象外

- `LICENSE-SCOPE.md` で `Apache-2.0` と整理したコード例・ツール・ビルド資産の通常利用
- `TRADEMARKS.md` に記載した商標利用許諾
- `THIRD_PARTY_NOTICES.md` に記載した third-party / upstream 資産

## 問い合わせ先

- Email: `knowledge@itdo.jp`
- GitHub Issue: `.github/ISSUE_TEMPLATE/commercial_license.md` を利用してください

## 連絡時に含めてほしい情報

- 利用主体（個人 / 法人 / 団体）
- 利用予定のコンテンツと範囲
- 利用媒体（Web、印刷、研修資料、動画等）
- 想定利用期間
- 連絡先
