# `shared/` の配置方針

`shared/` は、日本語版 (`src/ja/**`) と英語版 (`src/en/**`) の双方から参照する、言語非依存の共通資産だけを置くための領域です。

## `shared/` に置いてよいもの

- 両言語版で同一内容を再利用するコード例、テスト入力、データセット
- 文字列を埋め込まない図、もしくは別レイヤで各言語に差し替えられる図の元データ
- 共通テンプレート、変換用メタデータ、機械可読な補助ファイル
- 翻訳ではなく単一の canonical source を共有する必要がある資産

## `shared/` に置かないもの

- 本文、注釈、演習、章タイトルなど reader-facing な prose
- 言語別の front matter、ナビゲーション、ページ URL に依存する設定
- 日本語 UI / 英語 UI のスクリーンショットなど、ロケールが埋め込まれた画像
- 特定版だけで意味が完結する一時ファイルやレビュー用メモ

## 運用原則

- `shared/` に置く時点で、内容は `ja` / `en` のどちらにも従属しないものとして扱います。
- 言語依存の差分が必要になった場合は、`shared/` から移動し、各 edition 側で管理します。
- 共有対象を追加する PR では、どの edition から参照するかを説明してください。
- source-of-truth / translation / release の詳細は `BILINGUAL-WORKFLOW.md` を参照してください。
