## Acronyms

- **SAT**: Satisfiability
- **SMT**: Satisfiability Modulo Theories
- **LTL / CTL**: Linear Temporal Logic / Computation Tree Logic
- **CI**: Continuous Integration
- **RMW**: Read-Modify-Write

## Core Terms

- **Alloy**: A lightweight formal method centered on relational modeling and counterexample finding.
- **Invariant**: A property that must always hold.
- **Model Checking**: Exhaustive state-space exploration to determine whether a property holds.
- **Safety**: “Something bad never happens.”
- **Liveness**: “Something good eventually happens.”
- **TLA+**: A specification language for distributed and concurrent systems based on state transitions and temporal logic.
- **Z Notation**: A state-based formal specification language using schemas.
