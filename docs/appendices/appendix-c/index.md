# 付録C：記法対照表

Z 記法、Alloy、TLA+ などで共通に現れる概念の記号・用語を対照します（後続版で表形式に整備）。

| 概念 | Z 記法 | Alloy | TLA+ |
|---|---|---|---|
| 集合 | `A, B, ℙA` | `sig A {}`/`set A` | `SUBSET A` |
| 関係 | `R ⊆ A × B` | `A->B`（リレーション） | 関係集合で表現 |
| 関数 | `f: A ↔ B`/`A → B` | `fun A -> one B` 等 | 関数は写像として記述 |
| 不変 | `Inv`（スキーマ） | `fact` | `Invariant` |
| 状態更新 | `x'` | 仕様上の次状態関係 | `Next` / `x'` |
| 全称/存在 | `∀/∃` | `all/some` | `∀/∃` |

用語メモ
- スキーマ（Z）= 変数宣言＋制約の塊（構造化）
- fact（Alloy）= 常に成立すべき制約（不変）
- Invariant（TLA+）= すべての到達状態で成り立つ性質
