# 付録A：数学的基礎の復習

集合、論理、関数、関係、帰納・同値関係などを短く復習します。本文で用いる記法の最小限を対象にします（後続版で増補）。

## チートシート（最小セット）

- 集合: 空集合 `∅`、部分集合 `A ⊆ B`、差集合 `A \ B`、直積 `A × B`
- 関係: `R ⊆ A × B`、逆関係 `R⁻¹`、合成 `R ∘ S`
- 関数: 全/部分関数、単射/全射/全単射、写像 `f: A → B`
- 論理: 全称/存在 `∀, ∃`、含意 `→`、同値 `↔`、否定 `¬`
- 帰納法: 基底＋帰納ステップ、良基順序/変位関数
- 同値関係: 反射・対称・推移、商集合 `A/∼`

よく使う記号の読み替え（プログラミング）
- `∈`（属する）↔「メンバーである」、`⊆`（包含）↔「部分型/部分集合」
- 述語 `P(x)` ↔ 「ブール関数」、`¬P` ↔ `!P`
